# IoT-Integrated Wearable Stress Monitoring System

## Overview

This project is an IoT-based wearable stress monitoring system designed to detect stress levels in real time using physiological sensors.

## Features

* Real-time stress monitoring using GSR sensor
* Heart rate monitoring
* SMS alerts through GSM module
* GPS-based location tracking
* IoT cloud monitoring using Blynk
* Emergency buzzer alerts

## Hardware Components

* ESP32 Microcontroller
* GSR Sensor
* Heart Rate Sensor
* GSM Module
* GPS Module
* Buzzer
* Power Supply Unit

## Software Requirements

* Arduino IDE
* Embedded C
* Blynk IoT Platform

## Applications

* Personal health monitoring
* Stress detection
* Emergency alert systems
* Wearable healthcare devices
